## CHANGELOG

### v0.7.x

1. Added support for using controller names (from uhppoted.conf) in device commands. e.g. 
   ```
   uhppote-cli get-device alpha
   ```

2. Time profile support:
   -  `get-time-profile`
   -  `set-time-profile`
   -  `clear-time-profiles`
   -  `get-time-profiles`
   -  `set-time-profiles`

3. Documented commands missing from the README.
